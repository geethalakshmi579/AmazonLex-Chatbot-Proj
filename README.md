# Amazon Lex Chatbot Project

**Conversational AI Chatbot built using Amazon Lex**

## 📋 Project Overview
Built a functional chatbot using **Amazon Lex**, AWS’s fully managed service for creating conversational interfaces. The chatbot demonstrates natural language understanding and multi-turn conversation capabilities.

## ✨ Features Implemented
- Intent recognition and handling
- Slot filling for collecting user information
- Multi-turn dialog management
- Initial greeting responses and confirmation utterances
- Custom utterances for better intent detection

## 🛠️ Technology Used
- **Amazon Lex** (Primary & Only Service)
